# AI Spreadsheet Builder

`AI Spreadsheet Builder` is a Streamlit app that generates structured tabular data with OpenRouter, lets you review and edit the result in a grid, and exports a styled Excel workbook.

## 📱 App Link

**Try it here:** [https://ai-spreadsheet-builder.streamlit.app](https://ai-spreadsheet-builder.streamlit.app)



## Features

- Generate a dataset from a natural-language prompt
- Use OpenRouter models through the OpenAI-compatible API
- Review and edit data before export
- Apply a built-in Excel theme
- Optionally append a summary row for numeric columns
- Download the final workbook as `.xlsx`

## Project Structure

```text
AI-Spreadsheet-Builder/
|-- app.py
|-- ai_service.py
|-- config.py
|-- excel_formatter.py
|-- json_utils.py
|-- requirements.txt
|-- workbook_service.py
`-- README.md
```

- `app.py`: Streamlit UI flow and event handling
- `ai_service.py`: OpenRouter request logic and dataframe generation
- `config.py`: Shared constants and app configuration
- `excel_formatter.py`: Excel styling utilities
- `json_utils.py`: JSON extraction/parsing helpers
- `workbook_service.py`: Summary row logic and Excel export generation

## Requirements

Install the dependencies from `requirements.txt`:

```bash
pip install -r requirements.txt
```

## Run The App

From the `AI-Spreadsheet-Builder` directory:

```bash
streamlit run app.py
```

## How To Use

1. Enter your OpenRouter API key in the sidebar.
2. Choose an AI model.
3. Pick an Excel theme and decide whether to include a summary row.
4. Describe the dataset you want to generate.
5. Click `Generate`.
6. Review or edit the generated table.
7. Click `Compile and Format Final Excel Workbook`.
8. Download the generated Excel file.

## Notes

- The app expects the model to return JSON with `columns` and `data`.
- The editable table allows manual correction before export.
- Excel styling is applied locally in Python, not generated by the model.

## Example Prompt

```text
Create a table of 10 international markets for solar energy expansion with columns:
Country, Target MW, Entry Risk Index (1-10), Setup Cost Millions USD.
```
